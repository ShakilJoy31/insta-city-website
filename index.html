Difference between Local Storage and Session Storage!!

Session Storage

 The data we save or store save temporary. That means the data will be gone when we reload or close the tab. people or the developer use it less than using local storage. Data is not shared between all tabs and window from the same origin.

Local Storage

 The data is stored or save parmanently. That means when we reload or we close the tab the data we stored still remain unchanged. People or the developer use it more than the session storage. The data is used between all tabs and window from the same origin.
